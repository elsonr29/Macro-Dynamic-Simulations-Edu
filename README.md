# Macro-Dynamic-Simulations-Edu

Este projeto oferece um ambiente computacional para a simulação e análise de modelos macroeconômicos canônicos. O objetivo é integrar a teoria econômica à prática de programação (**R / Python / Julia**), permitindo que estudantes e pesquisadores visualizem dinâmicas de crescimento, ciclos econômicos e respostas a choques de política.

As implementações têm como base os modelos canônicos presentes nos principais manuais de macroeconomia e teoria do crescimento. O conteúdo serve como material de apoio para disciplinas de graduação, pós-graduação, cursos e mini-cursos.

---

## 🛠️ Sumário de Modelos

* **Product Variety Model (Romer):** Apresenta os fundamentos para a simulação do modelo de crescimento endógeno de inovação horizontal proposto por Romer. A estrutura da simulação segue a abordagem de Aghion e Howitt (2009, Cap. 5).
